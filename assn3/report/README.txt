1. 꼭 Release 모드로 빌드해주세요.
STL 사용율이 매우 높기 때문에 Debug 모드로 빌드 할 경우 실행이 매우 느립니다.

2. map.txt는 맵의 정보입니다.
첫 줄의 4 정수는 맵의 가로, 세로, 아이템의 개수, 적의 개수입니다.
수정하셔도 되지만 맵의 가장자리는 다 1(벽)으로 감싸주셔야 합니다.

3. 아이템을 현명하게 사용하세요.
여러개를 획득했더라도 획득된 순서대로 (FIFO) space를 눌러 사용할 수 있습니다.
아이템의 도움이 없으면 클리어가 사실상 불가능합니다.

4. 이동(방향키)는 꾹 눌러서 조작하고, 아이템(space)와 무기발사(왼클릭)는 press down만 인식하므로 여러번 사용하시려면 연타하여야합니다.

5. 게임이 종료되면 r을 눌러 다시시작하세요

6. v키를 누르면 시점을 1인칭과 3인칭끼리 전환할 수 있습니다.

7. resource폴더에 있는 것들은 런타임으로 읽혀지는 파일들로, 셰이더, 폴리곤 정보, 기타 config 파일 등이 있습니다.
